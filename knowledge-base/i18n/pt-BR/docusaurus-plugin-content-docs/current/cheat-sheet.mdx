# Cheat Sheet Rápido

Uma referência abrangente para desenvolvedores gerenciarem o monorepo **Elo Orgânico**, infraestrutura e automação.

---

## Executando a Plataforma

Utilizamos o **Turborepo** para orquestrar as aplicações e suas respectivas infraestruturas (bancos de dados/cache) em um único comando.

<Tabs>
  <TabItem value="instance" label="Instância de Comunidade" default>
    - **Iniciar Tudo:** `pnpm instance:dev` (Sobe infra + Web + API)
    - **Apenas Web:** `pnpm instance:web`
    - **Apenas API:** `pnpm instance:api`
    - **Parar e Limpar Tudo:** `pnpm instance:down`
  </TabItem>
  <TabItem value="portal" label="Portal Global">
    - **Iniciar Tudo:** `pnpm portal:dev` (Sobe infra + Web + API)
    - **Apenas Web:** `pnpm portal:web`
    - **Apenas API:** `pnpm portal:api`
    - **Parar e Limpar Tudo:** `pnpm portal:down`
  </TabItem>
  <TabItem value="docs" label="Documentação">
    - **Iniciar Docusaurus:** `pnpm docs:dev`
    - **Build da Docs:** `pnpm build` (Executa build para todos os workspaces, incluindo docs)
  </TabItem>
</Tabs>

## Infraestrutura (Docker)

Gestão direta dos serviços conteinerizados.

| Escopo | Comando | Descrição |
| :--- | :--- | :--- |
| **Instância** | `pnpm instance:up` | Inicia MongoDB (Replica Set) e Redis para Instância. |
| **Instância** | `pnpm instance:down` | Para e remove os containers e processos locais da Instância. |
| **Instância** | `pnpm instance:reset` | Apaga os dados e reinicia a infraestrutura da Instância. |
| **Portal** | `pnpm portal:up` | Inicia MongoDB (Replica Set) e Redis para o Portal. |
| **Portal** | `pnpm portal:down` | Para e remove os containers e processos locais do Portal. |
| **Portal** | `pnpm portal:reset` | Apaga os dados e reinicia a infraestrutura do Portal. |

## Gestão de Workspace

Lidando com dependências internas e catálogos.

- **Adicionar Dependência ao Catálogo:** `pnpm add <package> --catalog`
- **Adicionar Dependência a App Específica:** `pnpm add <package>@catalog: --filter @elo-instance/api`
- **Limpar node_modules:** `pnpm clean` (Usa `npkill`)
- **Validar Tipagem (Typecheck):** `pnpm typecheck`
- **Lint:** `pnpm lint`

---

## IA & Studio (MCP)

Orquestrando o Model Context Protocol e a ponte de design.

- **Penpot Studio:** `pnpm penpot:up` / `pnpm penpot:down`
- **Aide AI Bridge:** `pnpm penpot:aide:up` / `pnpm penpot:aide:down`
- **Busca Context7:** (Via Agente de IA) Pesquisa profunda em documentação técnica.

## Pipelines de Build

- **Build Completo:** `pnpm build`
- **Build Core (Instância):** `pnpm --filter @elo-instance/core build`
- **Build Core (Portal):** `pnpm --filter @elo-portal/core build`
